# Roadmap — Desatascos JF

## Estado actual

- Proyecto Astro creado.
- Tailwind CSS instalado.
- Estructura inicial creada.
- Home básica funcionando en local.
- Servicios confirmados añadidos:
  - Desatascos urgentes
  - Camión cuba
  - Limpieza de tuberías
  - Fontanería

## Principio principal

La web debe construirse como una versión de producción desde el inicio, aunque primero se enseñe al cliente como propuesta visual.

No se usarán frases como:

- Demo
- Prueba
- Texto provisional
- Lorem ipsum

No se añadirán servicios no confirmados por el cliente.
